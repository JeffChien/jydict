# Jydict

A command line tool for query vocabulary from yahoo dictionary.

# Installation
- `[sudo] npm install -g`

# basic usage
- `jydict <word or phrase>`
- `jydict` enter interactive mode, exit with `ctrl-c` or `ctrl-d`
- `jydict -h` display help
